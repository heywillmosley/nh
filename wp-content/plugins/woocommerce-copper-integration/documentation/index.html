A) About - top

The main task of this plugin is send your WooCommerce orders directly to your Copper CRM account.

B) Step 1 - Integration - top

Go to the WooCommerce -> Copper CRM.

Insert in the field `API token`. Your API key can be found/create under Settings > Preferences > API Keys. user guide

Insert in the field `User email`. Your CRM account email.

Save settings.

Configure status mapping and fields.

Save settings.
